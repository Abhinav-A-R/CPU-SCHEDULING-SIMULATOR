# CPU-SCHEDULING-SIMULATOR
A web-based OS simulator that visualizes CPU and disk scheduling algorithms (FCFS, SJF, Round Robin, Priority, etc.) and Banker’s Algorithm for deadlock avoidance. It provides Gantt charts, TAT, WT, and interactive analysis for better understanding of system behavior.
# 🚀 CPU Scheduling Simulator

A modern web-based simulator to visualize **CPU Scheduling, Disk Scheduling, and Banker’s Algorithm** with interactive UI, Gantt charts, and real-time analysis.

---

## 🌐 LIVE WEB FOR FREE
🔗 https://abhinav-a-r.github.io/cpu-scheduling-simulator/

---

## 📘 Description
CPU Scheduling Simulator is an interactive frontend web application built using HTML, CSS, and JavaScript. It helps students understand Operating System concepts through **visual simulations, dynamic charts, and performance analysis**.

---

## ✨ Features

### 🧠 CPU Scheduling Algorithms
- FCFS (First Come First Serve)
- SJF (Preemptive & Non-Preemptive)
- Priority Scheduling (Preemptive & Non-Preemptive)
- Round Robin

### 💾 Disk Scheduling
- Disk head movement visualization
- Seek time calculation
- Graphical disk chart representation

### 🔐 Banker’s Algorithm
- Deadlock avoidance simulation
- Safe and unsafe state detection
- Resource allocation visualization


### 📄 Page Replacement

- 📄 Supports multiple page replacement algorithms:
  - **FIFO (First In First Out)**
  - **LRU (Least Recently Used)**
  - **Optimal**

- ⚡ Real-time simulation of page replacement process

- 📊 Displays:
  - Page hits
  - Page faults
  - Hit ratio

- 🧠 Step-by-step visualization of frame allocation

- 🖥️ Simple and user-friendly interface

- 🔢 Custom input:
  - Number of frames
  - Page reference string

- 📈 Comparison between different algorithms

- 🔄 Dynamic updates without page reload (JavaScript based)

- 🎯 Educational tool for understanding OS concepts

### 📊 Visualization & Metrics
- Animated Gantt Charts
- Turnaround Time (TAT)
- Waiting Time (WT)
- Average TAT & WT
- Step-by-step execution view

### 🎨 UI/UX
- Dark / Light Mode 🌙☀️
- Responsive modern design
- Smooth animations and transitions
- Interactive input system
